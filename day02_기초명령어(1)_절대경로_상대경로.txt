탐색기		리눅스명령어	의미
-----------------------------------------------
←		cd -		이전 위치로
↑		cd ..		상위 폴더로
폴더더블클릭	cd [폴더이름]	지정한 폴더로 이동
주소창		pwd		현재 위치
파일목록		ls 		목록(간단히)
		ls -l		목록(자세히)

.	: 현재 위치
..	: 상위 위치
./	: 현재 디렉토리 안에 있는 어떤 대상
/	: 최상위 위치(디렉토리)

1) 경로에서 가장 앞에 사용되면 최상위 디렉토리
2) 경로 중간에 배치되면 특정 디렉토리 안에 있다는 의미

/	: 최상위 디렉토리
/var/www: 최상위 디렉토리 안에 있는 var 안에 있는 www

/windows/system32
C:\windows\system32

== 경로 ==

1) 절대 경로 : 어디서나 절대적인 위치를 참조하는 주소
2) 상대 경로 : 현재 위치에 따라 서로 다른 대상을 참조하는 주소

/ 로 시작하면 절대경로
/ 로 시작하지 않으면 상대경로
./ 로 시작하면 상대경로

pwd 는 항상 현재 위치를 절대경로로 출력한다

ls 는 대상을 지정하지 않으면 현재 위치를 대상으로 한다

※ 리눅스의 모든 디렉토리와 폴더는 이름의 첫글자가 .이면 숨김속성이다
